## Overview

Delta is a Windows-oriented Roblox script executor built for users who want a compact desktop utility for Lua injection and script execution. It pairs a simple interface with an included script hub, so Roblox scripts can be browsed and managed from a single place while keeping the workflow quick and practical.

In addition to straightforward execution, Delta provides a saved script queue, batch execution, a built-in debugger, and auto-update support. These additions are meant for people looking for a Roblox Lua executor with a more integrated routine rather than separate tools for finding, loading, and running scripts.
